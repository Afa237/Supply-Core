\# Supply Core

Supply Core is a modern web-based Supply Chain Management (SCM) platform designed to help businesses manage procurement, inventory, warehouses, suppliers, logistics, and reporting from a single dashboard.

\## Tech Stack

* Python

* Django

* Bootstrap 5

* JavaScript

* PostgreSQL

\## Planned Features

* User authentication and role management

* Supplier management

* Product and category management

* Inventory tracking

* Warehouse management

* Procurement workflows

* Shipment and logistics tracking

* Reports and analytics

* Notifications and alerts

\## Project Status

🚧 Currently in active development.

This repository documents the complete journey of building Supply Core from planning and database design to deployment and production release.
